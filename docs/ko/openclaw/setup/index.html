DOCS / OPENCLAW

OpenClaw를 연결하고 실행은 Koinara-node에 맡기세요.

OpenClaw는 에이전트 계층이고, Koinara-node는 프로토콜 실행 경계입니다. 노드 등록, heartbeat, provider 제출, verifier 참여, 보상 claim은 계속 Koinara-node가 맡습니다.

1단계부터 시작 설치 가이드 저장소 문서

순서

OpenClaw는 추론, Koinara-node는 실행을 담당합니다.

Step 1

로컬 OpenClaw CLI 응답 확인

내장 backend는 같은 CLI 경로를 그대로 사용합니다.

 Step 2

번들된 OpenClaw 스킬 설치

이제 저장소 안에 Koinara 전용 스킬 패키지가 포함됩니다.

 Step 3

OpenClaw provider 경로 실행

doctor, start, claim 전용 명령을 사용하면 됩니다.
01

STEP 1

먼저 OpenClaw가 로컬에서 응답하는지 확인하세요.

openclaw agent --agent main --local --json --message "Reply with exactly OK"
02

STEP 2

번들된 OpenClaw 스킬 패키지를 설치하세요.

powershell -ExecutionPolicy Bypass -File .\scripts\install-openclaw-skill.ps1
  • 복사 위치: %USERPROFILE%\.openclaw\skills\koinara-node
  • OpenClaw가 이미 열려 있으면 재시작하거나 스킬을 다시 불러오세요.
  • 이 저장소를 OpenClaw 워크스페이스로 열면 저장소 내부 skill도 바로 사용할 수 있습니다.
03

STEP 3

OpenClaw provider 전용 명령으로 doctor, start, claim 하세요.

npm.cmd run provider:v2:openclaw:doctor
npm.cmd run provider:v2:openclaw:start
npm.cmd run provider:v2:openclaw:claim
