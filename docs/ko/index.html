Koinara는 오직 두 가지 프로토콜 기본값만 정의합니다. Minimum Acceptable Inference (MAI)와 Minimum Reward입니다. 그 기준 위에 있는 품질, 속도, 깊이, 사용자 선호는 시장의 영역으로 남겨둡니다.

프로토콜 바닥선

정식으로 규정되는 것은 최소 기준뿐입니다.

Minimum Acceptable Inference

MAI는 최고의 정답을 보장하는 오라클이 아니라 최소 수용선을 정의하는 기준입니다.

Minimum Reward

KOIN 발행은 프로토콜 바닥선을 충족한 추론에만 지급되며, premium reward는 시장 회계로 분리됩니다.

Proof of Inference

PoI는 제출물이 최소 프로토콜 조건을 충족했다는 사실을 증명하지만, 절대적 진실을 보증하지는 않습니다.

공개 노드

프로토콜은 여기서 읽고,
노드는 별도 레포에서 실행합니다.

Koinara-node

provider, verifier, 또는 둘 다

`Koinara-node`는 지원 네트워크에 배포된 Koinara 컨트랙트에 직접 연결되는 독립형 공개 노드입니다.

레포 열기 홈 테스트 빠른 시작
설치

3단계 운영자 경로

git clone https://github.com/sinmb79/Koinara-node.git
cd Koinara-node
npm install
npm run setup
npm run doctor
npm run node
문서

노드 패키지에 포함된 내용

  • provider / verifier / both 역할
  • ollama / openai 백엔드
  • OpenClaw를 로컬 노드 위의 에이전트 계층으로 연결 가능
  • priority failover 또는 all-healthy 멀티체인 선택
  • doctor, status, logs 명령
  • manifest + receipt discovery spec
  • 홈 테스트와 운영자 체크리스트
노드 설정 OpenClaw 연동 운영자 체크리스트 지원 네트워크
