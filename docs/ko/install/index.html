DOCS / WINDOWS 설치

Koinara-node를 Windows에서 바로 설치하세요.

이 페이지는 처음 설치하는 운영자를 위한 실제 Windows 경로입니다. PowerShell의 npm.ps1 오류를 피하고, 저장소는 사용자 폴더 아래에 두며, Worldland v2 명령만 따라가면 되도록 정리했습니다.

1단계부터 시작 OpenClaw 연결 가이드 저장소 문서

순서

아래 순서대로 따라 하세요.

Step 1

사용자 폴더 아래에 클론

Desktop이나 system32에서는 시작하지 마세요.

 Step 2

setup을 실행하고 홈 경로 기본값 유지

런타임 파일은 %USERPROFILE%\.koinara-node 아래에 저장됩니다.

 Step 3

doctor, start, claim 순서로 실행

라이브 Worldland v2 명령을 그대로 사용하면 됩니다.
01

STEP 1

현재 로그인한 사용자 폴더 아래에 저장소를 클론하세요.

cd $env:USERPROFILE
git clone https://github.com/sinmb79/Koinara-node.git koinara-node
cd $env:USERPROFILE\koinara-node
npm.cmd install
Windows PowerShell에서는 npm 대신 npm.cmd를 쓰는 편이 안전합니다. 그래야 npm.ps1 실행 정책 오류를 피할 수 있습니다.
02

STEP 2

setup을 실행하고 기본 경로를 그대로 사용하세요.

npm.cmd run setup
  • manifest / receipt root: %USERPROFILE%\.koinara-node\network
  • artifact root: %USERPROFILE%\.koinara-node\artifacts
  • state root: %USERPROFILE%\.koinara-node\state
  • 지갑 키와 RPC 비밀값은 로컬 파일에만 저장하세요.
03

STEP 3

doctor로 확인하고, start 후 epoch가 닫히면 claim하세요.

npm.cmd run provider:v2:doctor
npm.cmd run provider:v2:start

# verifier도 같이 돌릴 경우 다른 PowerShell 창에서
npm.cmd run verifier:v2:doctor
npm.cmd run verifier:v2:start

# 현재 epoch가 닫힌 뒤에만 claim
npm.cmd run provider:v2:claim
npm.cmd run verifier:v2:claim
