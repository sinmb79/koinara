이 페이지 읽는 법

active reward는 종료된 epoch의 활성 노드들이 균등하게 나눕니다. work reward는 accepted job 기준으로 계산되고, provider 70%, verifier 집합 30%로 나뉩니다.

epoch가 아직 열려 있으면 예상 보상 경로만 보이고, 실제 잔액은 아직 민팅되지 않습니다. epoch 종료 후 claim 트랜잭션을 보내야 지갑에 KOIN이 발행됩니다.
