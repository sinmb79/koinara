## Notes

- The Markdown documents are the editable source texts.
- The PDF files are the distribution-ready exports.
- The main whitepaper now stays concise while deployment profiles, portability notes, and parameter annexes live in separate appendix documents.
- The whitepaper includes protocol formulas, state-machine figures, architecture figures, and issuance-curve visuals.
- If the protocol text changes, update both language versions and companion appendices, then regenerate all PDFs with `python tools/render_whitepaper_pdfs.py`.

## Koinara at a Glance

- Official name: `Koinara: A Peer-to-Peer Collective Inference Network`
- Tagline: `The open network for collective inference`
- Core protocol primitives: `Minimum Acceptable Inference (MAI)` and `Minimum Reward`
- Protocol scope: minimum validity and minimum issuance, not subjective answer quality
