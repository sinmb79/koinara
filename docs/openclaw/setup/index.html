01

STEP 1

Verify that OpenClaw responds locally.

openclaw agent --agent main --local --json --message "Reply with exactly OK"
Do not continue until this command returns a valid JSON payload. The Koinara provider backend shells out to this exact CLI path.
02

STEP 2

Install the bundled OpenClaw skill package.

powershell -ExecutionPolicy Bypass -File .\scripts\install-openclaw-skill.ps1
  • The skill is copied into %USERPROFILE%\.openclaw\skills\koinara-node.
  • If OpenClaw already runs, restart it or reload skills.
  • If you open this repo as an OpenClaw workspace, the repo-local skill can also be used directly.
03

STEP 3

Run the OpenClaw-backed provider commands.

npm.cmd run provider:v2:openclaw:doctor
npm.cmd run provider:v2:openclaw:start
npm.cmd run provider:v2:openclaw:claim

What OpenClaw does

Generates the provider-side response text through the local agent CLI.

What Koinara-node does

Registers the node, sends heartbeat, hashes the response, submits on-chain, and claims rewards.
