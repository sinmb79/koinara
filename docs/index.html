MINIMUM PROTOCOL / OPEN INFERENCE MARKET

The open network for collective inference.

Koinara defines only two protocol primitives: Minimum Acceptable Inference (MAI) and Minimum Reward. Everything above that floor belongs to the market.

PROTOCOL FLOOR

Only the minimum is canonical.

Minimum Acceptable Inference

MAI is not a best-answer oracle. It is the minimum line of protocol acceptance, defined in v1 by valid job state, deadline compliance, non-empty response, format pass, and verifier approval.

Minimum Reward

KOIN issuance rewards accepted inference at the protocol floor. Premium rewards stay market-funded and must remain separate from protocol issuance.

Proof of Inference

PoI proves that a submission satisfied the minimum protocol conditions. It does not prove universal truth or market-best quality.

PUBLIC NODE

Read the protocol here. Run the node from its own repo.

Provider, verifier, or both

Koinara-node is the standalone public background node for the network. It connects directly to deployed Koinara contracts across supported networks, with no hosted API, no business layer, and no UI dependency.

Open Repo Quickstart
terminal
$ git clone https://github.com/sinmb79/Koinara-node.git $ cd Koinara-node $ npm install $ npm run setup $ npm run doctor $ npm run node

Three-step operator path — clone, setup, run.

  • ollama / openai backends
