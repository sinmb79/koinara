PUBLIC NODE

Run the open node. Earn from uptime and accepted work.

Standalone public runtime

Koinara-node is the public provider/verifier runtime. It talks directly to deployed contracts and does not depend on a hosted API or business service.

Open Repo Node Setup
terminal
$ git clone https://github.com/sinmb79/Koinara-node.git $ cd Koinara-node $ npm install $ npm run provider:v2:start $ npm run verifier:v2:start

v2 commands target the live Worldland deployment and keep state in separate v2 folders.

What the v2 node path adds

  • provider / verifier role-specific v2 entrypoints
  • automatic node registration and heartbeat
  • active reward claim path
  • deferred work reward claim path
  • separate v2 mainnet config and artifact directories
