独立公开运行时

Koinara-node 是公开的 provider / verifier 运行时，不依赖托管 API 或业务层，直接连接已部署合约。

打开仓库 节点文档 OpenClaw 路径
terminal
$ git clone https://github.com/sinmb79/Koinara-node.git $ cd Koinara-node $ npm install $ npm run provider:v2:start $ npm run provider:v2:claim $ npm run verifier:v2:start

v2 命令会使用 Worldland 主网 v2 路径与独立的 v2 状态目录， epoch 结束后可以再用 claim 专用命令领取 KOIN。

v2 节点路径新增能力

  • provider / verifier 的 v2 专用入口
  • epoch 结束后的 claim 专用命令
  • 自动节点注册与 heartbeat
  • active reward claim 路径
  • epoch 结束后的 work reward claim 路径
  • 使用本地 OpenClaw CLI 的内建 provider backend
  • 独立的 v2 mainnet 配置与产物目录
