Koinara 只定义两个协议原语： Minimum Acceptable Inference (MAI) 与 Minimum Reward。高于这条底线的质量、速度、深度与用户偏好，都属于市场层。

协议底线

被正式定义的只有最低标准。

Minimum Acceptable Inference

MAI 不是最佳答案预言机，而是协议接受的最低门槛。

Minimum Reward

KOIN 发行只奖励达到协议底线的推理，premium reward 仍由市场承担。

Proof of Inference

PoI 证明提交满足最低协议条件，但不证明绝对真理或市场最佳质量。

Koinara-node

provider、verifier，或两者兼有

`Koinara-node` 是连接已部署 Koinara 合约的独立公开后台节点程序，适用于多链运行时。

打开仓库 家庭测试快速开始
安装

三步操作路径

git clone https://github.com/sinmb79/Koinara-node.git
cd Koinara-node
npm install
npm run setup
npm run doctor
npm run node
文档

节点包包含的内容

  • provider / verifier / both 角色
  • ollama / openai 推理后端
  • priority failover 或 all-healthy 多链选择
  • doctor、status 与 logs 命令
  • manifest + receipt 发现规范
  • 家庭测试与操作员检查清单
