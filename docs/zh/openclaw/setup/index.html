连接 OpenClaw， 把执行留给 Koinara-node。

OpenClaw 是代理层，Koinara-node 仍然负责协议执行： 节点注册、heartbeat、provider 提交、verifier 参与与奖励 claim。

Step 1

确认本地 OpenClaw CLI 可以返回结果

 Step 2

安装仓库附带的 OpenClaw 技能包

 Step 3

运行 OpenClaw provider 专用命令
01

先验证本地 OpenClaw agent。

openclaw agent --agent main --local --json --message "Reply with exactly OK"
02

安装仓库附带的 OpenClaw 技能包。

powershell -ExecutionPolicy Bypass -File .\scripts\install-openclaw-skill.ps1
03

运行 OpenClaw provider doctor、start 和 claim。

npm.cmd run provider:v2:openclaw:doctor
npm.cmd run provider:v2:openclaw:start
npm.cmd run provider:v2:openclaw:claim
