DOCS / WINDOWS 安装

在 Windows 上安装 Koinara-node。

这是一条给首次运营者使用的 Windows 路径。它会避开 PowerShell 的 npm.ps1 错误，把仓库放到当前用户目录下，并直接使用 Worldland v2 的命令。

从步骤 1 开始 OpenClaw 指南
Step 1

在用户目录下克隆仓库

不要把仓库放在 Desktop 或 system32 下。

 Step 2

运行 setup 并保留默认路径

运行时文件位于 %USERPROFILE%\.koinara-node。

 Step 3

执行 doctor、start 和 claim

使用 Worldland v2 的专用命令。
01

先在当前用户目录下克隆仓库。

cd $env:USERPROFILE
git clone https://github.com/sinmb79/Koinara-node.git koinara-node
cd $env:USERPROFILE\koinara-node
npm.cmd install
02

运行 setup，并保留默认的用户主目录路径。

npm.cmd run setup
03

先运行 doctor，再 start，epoch 结束后再 claim。

npm.cmd run provider:v2:doctor
npm.cmd run provider:v2:start
npm.cmd run provider:v2:claim
