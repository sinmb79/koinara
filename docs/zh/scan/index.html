如何理解本页

active reward 会在已关闭 epoch 内，由所有 active node 平均分配。 work reward 按 accepted job 计算，然后按 provider 70%、verifier 集合 30% 拆分。

如果 epoch 仍然开放，你会看到预计奖励路径，但余额还不会立即变化。 必须等 epoch 关闭后发送 claim 交易，KOIN 才会真正铸造到钱包。
