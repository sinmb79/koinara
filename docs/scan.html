How to read this page

Active rewards are split evenly among active nodes in a closed epoch. Work rewards belong to accepted jobs and are split 70/30 between provider and verifier set.

If an epoch is still open, you may see an expected reward path without minted balance yet. Minting happens when the node wallet sends the claim transaction after the epoch closes.
